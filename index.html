Polymarket Event Explorer

Fetch condition IDs for sports events — moneyline, totals, spreads, and BTTS. Use a /market/... URL for one market, or a sports/event URL to load all related markets.

Also accepts /market/fifwc-...-total-5pt5 URLs, raw slugs, or condition IDs (0x...). Gamma only returns moneyline on /events — this tool also probes totals, spreads & BTTS slugs.
